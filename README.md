
# AMITASKER

Frond end Template for social media management company

## Installation

Use the package manager [npm](https://nodejs.org) to install Dependency. Used `node-sass` to compile sass.

```JS
npm install
```

or

```JS
yarn install
```

## Usage

for development

```JS
npm run dev
```

for building

```JS
npm run dev
```

## dependency

you will found on `/assets/vendor/`

* Bootstrap 4
* modernizr
* hover CSS
* fontawesome
* Prefixfree
* jQuery
* linearicons
